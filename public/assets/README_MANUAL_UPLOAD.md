# Custom Media Asset Uploder

Use this folder to place your custom rich-media background images and video files. The application is pre-configured to automatically load them from these exact paths if present.

### Expected Files:
1. **`mycoffe1.jpg`** — Main Hero Visual / Dashboard Core background image.
2. **`mycoffe2.jpg`** — Secondary ambient coffee details (espresso with nice crema, high-quality, 1:1 or 4:3).
3. **`mycoffe3.jpg`** — Coffee routines and wellness habits illustration card background (4:3).
4. **`mycoffe4.mp4`** — Ambient video loops to create a cozy cafe study environment.

---
### Manual Upload Instructions:
- Drop your jpeg or mp4 files in this folder using the file explorer on the left or system uploader.
- Make sure they are named exactly as listed above (match spelling and casing).
